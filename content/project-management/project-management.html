Mercurial Tips on Mac OS X

If you're using Mercurial on Mac you probably already noticed case-folding collisions are always a problem.

$ hg update abort: case-folding collision between somefile and SomeFile

That's because Mac OS X and Windows, oposite to Linux and other OS's, aren't case-folding sensitive. Since Mercurial 1.7.5 this problem seems to be more or less solve on Windows but it's still a problem on Mac.

The easiest solution is to setup the Mercurial Case Fold Extension. Here are the steps you should follow:

Download the Extension on your Desktop. $ cd ~/Desktop $ hg clone https://bitbucket.org/bradobro/hgfold Copy the Extension to your libraries directory. $ sudo cp -r hgfold /usr/local/bin/hgfold Go to your mercurial repository root. $ cd ~/your_repository_folder Edit the your repository hgrc file $ open .hg/hgrc // you can also edit it with Nano or Vim On extensions add the path to your extension [extensions] fold = /usr/local/lib/hgfold/fold.py Done it! Now every time Mercurial shows you a case-collision problem just type. $ hg update --fold
