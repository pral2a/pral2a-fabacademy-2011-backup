Serial to RGB BOARD! You type a color and...

The board has a preloaded list of 24 colors associated with its 24-bit RGB representation (0-255 x 0-255 x 0-255). The board can receive serial messages coming from your serial terminal. When a message match iany of the colors avaiable on the preloaded color list the board does Soft PWM on the 3 pins the RGB LED is conectected. If it's succesfull it returns a message over the serial port with the color.
