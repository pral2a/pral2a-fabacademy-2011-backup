THE PROGRAMMING

The small app is written in Processing. The serial interface with the network is done using the Serial library. The data is published to Pachube using the EEML (Extended Environments Markup Language) that uses the XML data modeling structure, this is implemented using the EEML library . The GUI is based on the ControlP5 library.

The app is based on diferent modules.

  • 1. Network receiver, where the nodes comming from the serial port are check and unpackage.
  • 2. Network Buffer, where data comming from the nodes is stored waiting to be processed.
  • 3. Pachube, where data is publish to Pachube.
  • 4. GUI, the user interface for visualizing the nodes data and setup the parameters.
  • 5. Configuration setup, where all the parameters are set depending on the setup.
  • 6. File Reader / Writer, manage the log and config text files.
